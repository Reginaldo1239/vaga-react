# Ecommerce

desenvolvido usando react,
imagens hospedada no s3

## roda o projeto

clone o rep√≥sitorio

```bash
 git clone https://github.com/Reginaldo1239/vaga-react.git

```
na pasta ecommerce/src ,inicie o json-server
```bash
json-server --watch db.json --port 3001

```

na pasta ecommerce
```bash
 npm install
 npm start
```
